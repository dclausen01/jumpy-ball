# Jumpy Ball

A simple, interactive "Flappy Bird" style game built using HTML5 Canvas and JavaScript.

## Features
- **Dynamic Difficulty:** The gap size and colors change as the level progresses.
- **Responsive Design:** Adapts to window resizing.
- **Touch Support:** Playable on mobile devices.
- **Visuals:** Simple, colorful graphics featuring clouds, ground, and a moving bird.

## How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/jumpy-ball.git
   ```
2. Open `index.html` in your favorite web browser.

## Controls
- **Space** or **Arrow Up**: Jump / Restart
- **Touch Screen**: Jump / Restart on mobile devices

## Development Info
This project was completely engineered using the **gemma4:12b** model and the **opencode** CLI tool. 🚀
